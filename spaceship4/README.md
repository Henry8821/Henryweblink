## 🚀 How to Put This Game Online (Step by Step)

### Option A — Drag & Drop Upload (easiest, no Git needed)

1. Go to [github.com](https://github.com) and sign in
2. Click **"New repository"**, name it (e.g. `nova-strike`), set it to **Public**, click **Create**
3. On the empty repo page, click **"uploading an existing file"**
4. Open your game folder on your computer, **select ALL files and folders** and drag them into the GitHub upload box
5. Scroll down, click **"Commit changes"**
6. Go to **Settings → Pages**
7. Under **Source**, select **Deploy from a branch**
8. Set branch to **main**, folder to **/ (root)**, click **Save**
9. Wait ~60 seconds, then your game is live at:
   `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

### Option B — GitHub Desktop (if you have it installed)

1. Open GitHub Desktop → **File → Add Local Repository** → select this folder
2. If it says "not a git repo", click **"create a repository"**
3. Click **Publish repository** (make sure **Keep this code private** is **unchecked**)
4. Then follow steps 6–9 from Option A above

## 🎮 Features

- 5 levels with increasing difficulty
- 4 playable ships with unique stats  
- 7+ power-ups (shield, rapid fire, nova bomb, magnet, and more)
- Boss fights on levels 3 and 5
- Weapon shop with unlockable weapons
- Achievements, leaderboard, and rewards system
- Fully offline — no server needed, runs in any modern browser
