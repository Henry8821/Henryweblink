# NOVA STRIKE 🚀

A high-octane browser space shooter — dodge asteroids, collect power-ups, and fight bosses.

## Play Now

👉 **[Play NOVA STRIKE](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/assets/www/index.html)**

> Replace the link above with your actual GitHub Pages URL after enabling it in repo Settings → Pages.

## Features

- 5 levels with increasing difficulty
- 4 playable ships with unique stats
- 7+ power-ups (shield, rapid fire, nova bomb, magnet, and more)
- Boss fights on levels 3 and 5
- Weapon shop with unlockable weapons
- Achievements, leaderboard, and rewards system
- Fully offline — no server needed, runs in any modern browser

## How to Host on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your game will be live at `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move | Arrow keys / WASD | On-screen joystick |
| Fire | Space / Z | Fire button |
| Pause | P / Escape | Pause button |

## Tech Stack

Pure HTML5 Canvas + Vanilla JavaScript. No frameworks, no build step — just open `assets/www/index.html` in a browser.
